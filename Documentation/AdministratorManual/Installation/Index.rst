Installation
============
The extension needs to be installed as any other extension of TYPO3 CMS:

Install Extension from EM:
--------------------------

#. Switch to the module “Extension Manager”.

#. Get the extension

#. **Get it from the Extension Manager:** Press the “Retrieve/Update” button and search for the extension key *ns_instagram* and import the extension from the repository.

#. **Get it from typo3.org:** You can always get current version from `https://extensions.typo3.org/extension/ns_instagram/ <https://extensions.typo3.org/extension/ns_instagram/>`_ by downloading either the t3x or zip version. Upload the file afterwards in the Extension Manager.

.. figure:: Images/Typo3-NsInstagram-install-1.png
  :width: 800px
  :alt: Install_1


Preparation: Include static TypoScript
--------------------------------------

The extension ships some TypoScript code which needs to be included.

#. Switch to the root page of your site.

#. Switch to the **Template module** and select *Info/Modify*.

#. Click the link **Edit the whole template record** and switch to the tab *Includes*.

#. Select **[NITSAN] ns_instagram** at the field *Include static (from extensions):*

.. figure:: Images/Typo3-NsInstagram-install-2.png
  :width: 800px
  :alt: Install_2
